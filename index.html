Static Website • CloudFront • SSL • CodePipeline

📘 About This Project

This site auto-deploys via CodePipeline from GitHub to S3, secured with ACM and distributed globally using CloudFront. No manual deploys — just push and ship.

⚙️ Tech Stack

  • Hosting: Amazon S3 (Static Website)
  • CI/CD: AWS CodePipeline (GitHub → S3)
  • SSL: ACM for HTTPS
  • CDN: CloudFront with cache invalidation

📈 Why This Matters

It’s a production-ready pipeline using modern cloud architecture. Fully automated and scalable — ideal for dev teams or solo projects alike.
